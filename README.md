# 3MIN

Modern anonymous real-time chat platform built with Next.js and Socket.IO.

3MIN focuses on fast, meaningful conversations through smart matching based on interests and conversation vibe.

---

## Features

- Real-time anonymous chat
- Smart matching system
- Interest-based pairing
- Vibe matching
- 3-minute countdown chats
- Skip & auto-rematch
- Typing indicators
- Match compatibility percentage
- Reveal identity system
- Icebreaker prompts
- Mobile responsive UI
- Modern clean interface

---

## Tech Stack

### Frontend
- Next.js 16
- React 19
- Tailwind CSS
- Lucide React Icons

### Backend
- Node.js
- Socket.IO

---

## Installation

Clone the repository:

```bash
git clone  https://github.com/KgosiS/3min.git
```

Move into the project:

```bash
cd 3min
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
